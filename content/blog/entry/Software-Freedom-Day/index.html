This Saturday 10th September is international Software Freedom Day, a day dedicated to educating the public about the wholesome goodness of free-as-in-freedom software. There are quite a few events being held in Australia, with the Melbourne event being run by Linux Users of Victoria and Computerbank Victoria.

If you want to learn more about the amazing range of quality software products available without charge from the open-source community, track down the LUV members in the Melbourne CBD for a chat and a free software CD.
